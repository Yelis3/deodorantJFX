Analizador de malos olores realizado por Juan Pablo Ovalle Sánchez y Yeliana Andrea Torres Medina,
para la materia de Lenguajes de Programación dictada por el profesor Felipe Restrepo Calle.

El programa analiza, para el lenguaje de programación Python, tres malos olores de código,
para dos de los cuales proporciona el refactor correspondiente, junto a una explicación del por qué es conveniente hacer refactor.
Ésto dejandolo a criterio del usuario qué malos olores desea correguir.

Para la compilación y ejecución se recomienda el uso de INTELLIJ IDEA, abriendo desde él, la carpeta raiz del proyecto.

El archivo con los casos de prueba se encuentra en la carpeta raiz del proyecto, bajo el nombre de inputTest.txt.
